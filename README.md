# Инструкция по использованию системы детектирования саженцев
## Обзор системы
Система предназначена для автоматического подсчета саженцев пихт на фотографиях. Состоит из трех основных скриптов:

active_light.py	— автоматическая подстройка под освещение
opencv.py — базовое детектирование с фиксированными параметрами
overlay.py — объединение двух половинок панорамного изображения

# 1. active_light.py — Детектирование с автонастройкой
Автоматически анализирует яркость изображения и подбирает оптимальные параметры для каждого снимка, затем считает количество саженцев.

Анализ яркости — изображение переводится в HSV пространство, вычисляется средняя яркость (0–255)

Классификация освещения:

+ very_dark — очень темное (<100)
+ dark — темное (100–130)
+ normal — нормальное (130–180)
+ bright — яркое (>180)

Подбор параметров — для каждого типа выбираются свои настройки:
Диапазоны оттенков (Hue), насыщенности (Saturation), яркости (Value)

Параметры морфологической обработки

Детектирование — выделение зеленых областей, фильтрация шума, подсчет контуров

Результат
Откроются окна с обработанными изображениями (отмеченные саженцы)

В консоль выводится информация о яркости и количестве саженцев для каждого файла

Создается файл results.txt с итогами

Управление
Нажмите ESC для закрытия всех окон и завершения

2. opencv.py — Базовое детектирование
Что делает
Детектирует саженцы на ОДНОМ изображении с фиксированными параметрами (ручная настройка).

Настройка параметров
В коде можно изменить параметры функции detect_seedlings():

python
detect_seedlings(
    image_path="test1.jpg",   # путь к изображению
    h_min=20, h_max=85,       # Оттенок (зеленый)
    s_min=100, s_max=255,     # Насыщенность
    v_min=110, v_max=255,     # Яркость
    min_area=50,              # Минимальная площадь объекта
    erode_iter=1,             # Эрозия (удаление шума)
    dilate_iter=2,            # Дилатация (заполнение дырок)
    blur_size=5               # Размытие
)
Что отображается
Окно	Содержимое
«Original»	Исходное изображение с прямоугольниками вокруг саженцев
«Mask before»	Маска ДО морфологической обработки (что выделилось по цвету)
«Mask after»	Маска ПОСЛЕ обработки (отфильтрованные объекты)
Результат
Количество саженцев выводится в консоль

Изображения отображаются в окнах

3. overlay.py — Объединение панорамы
Что делает
Объединяет две половинки панорамного снимка в одно целое изображение с автоматическим выравниванием.

Как это работает
Загрузка двух изображений (half1.jpg и half2.jpg)

Алгоритм ORB находит характерные точки на обоих снимках

Сопоставление точек для определения вертикального смещения

Объединение изображений с найденным смещением

Обрезка пустых областей

Подсветка зоны наложения

Входные файлы
Файл	Описание
half1.jpg	Первая половинка панорамы
half2.jpg	Вторая половинка панорамы
Результат
Файл	Описание
result.jpg	Объединенное изображение
result_highlighted.jpg	То же изображение с подсвеченной зоной наложения (синим)
Выходные данные в консоли
Найденное смещение между изображениями

Размер итогового изображения

Область наложения в пикселях
